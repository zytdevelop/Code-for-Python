## 可迭代对象
- 可以更新迭代的实实在在的值
- 内部含有 '__iter__'方法
- str, list, tuple, dict, set, range
- 优点：操作方法多，操作灵活，直观
- 缺点：占用内存
- 可迭代对象通过调用iter()函数转换为迭代器   
```python
a = list()
b = iter(a)  # b就是迭代器
```

## 迭代器
- 可以更新迭代的数据结构(工具
- 内部含有 '__iter__'方法和'__next__'方法
- 本质上可以理解为文件句柄
- 优点：节省内存
- 缺点：不直观，速度相对慢，操作方法单一，不可回头
- 迭代器总是可迭代的
